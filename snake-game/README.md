# Retro Snake - CRT Edition

A classic Snake game with a retro CRT monitor aesthetic. Built with HTML5 Canvas, CSS, and Vanilla JavaScript.

## Features
- **Retro CRT Aesthetic**: Scanlines, flicker effects, and green phosphor glow.
- **Classic Gameplay**: Navigate the snake to eat food and grow longer.
- **High Score**: Keeps track of your best score using `localStorage`.
- **Chiptune Sounds**: Retro-style audio generated via Web Audio API.
- **Responsive Controls**: Use arrow keys for movement.

## How to Play
1. Open `index.html` in any modern web browser.
2. Press **Space Bar** to start the game.
3. Use **Arrow Keys** to change direction.
4. Eat the green pixels to grow and increase your score.
5. If you hit the wall or yourself, it's Game Over!
6. Press **Space Bar** to restart after a game ends.

## Technologies Used
- HTML5 Canvas
- Vanilla JavaScript
- CSS (Gradients, Animations, Filters)
- Web Audio API (for chiptune sounds)
- Google Fonts (Press Start 2P)

## Rules
- Snake moves in one direction at a time.
- Direction can be changed using Up, Down, Left, and Right arrow keys.
- Food appears randomly on the board.
- Snake grows when it eats food.
- Game ends when the snake hits the wall or its own tail.
- Score increases by 10 for every food item eaten.
